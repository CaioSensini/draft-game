# Draft Game - Foundation

## Stack escolhida
- Game Client: HTML5 + TypeScript + Phaser
- Desktop: Electron
- Mobile: Capacitor
- Backend: NestJS
- Banco: PostgreSQL
- Admin: Next.js

## Estrutura
- game-client: jogo
- backend_api: API
- admin_panel: painel
- docs: documentação
- assets_base: assets

## Direção atual do protótipo
- Arena 16x6
- Combate 4x4
- Classes finais: Rei, Guerreiro, Especialista, Executor
- Core atual focado em combate, UX e base para evolução futura
